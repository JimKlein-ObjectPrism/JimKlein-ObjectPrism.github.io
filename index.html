Kartini Meal Plan Food Journal

Jim Klein

Welcome to GitHub Pages.

It helps support a patient’s ordered eating by streamlining reporting for parents of patients. It also can help in building the habits of a lifetime of ordered eating for patients by providing a way to track meals after leaving treatment.

This release of the app contains these features:

  • Home screen: Time-of-day aware quick link to the appropriate meal.
  • Track screen: Access, enter, and edit all meals for the current day and any previous days.
  • Print screen: Print out the food journal log entry for a selected day. Or use the print preview to quickly review details for any day.
  • Profile screen: Enables setup of patient, parents, required snacks, Medicines, and Addons. This version of the App comes set up with a placeholder patient name and parent names.

    • Home Screen

    The home screen has a context sensitive Log Meal button that takes the parent to the current meal in a single click. The status of the current meal is also displayed.
